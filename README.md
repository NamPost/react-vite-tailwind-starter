# React Vite Boilerplate

A template/boilerplate for rapid application development @NamPost. 

* `Tailwind CSS`
* `Headless UI`
* `React-Query`
* `Tanstack Router`
* `Jotai State`

### Getting Started

Clone the repo.

    npm i
    npm run dev
